# SpaceX launches API with client

Original SpaceX API could be found [here](https://github.com/r-spacex/SpaceX-API)

## How to install

- `npm run install`

## How to build client

- `npm run build:client`

## How to run

- `npm run start:server` starts server
- `npm run start:client` starts client

## Requirement

- `node v18.3.0`
